# EatClub Challenge

## Setup Instructions

To run this project locally:

1. Clone the project from repository: `https://github.com/Gbanya-Michael/eatclub-challenge-michael`

```bash
git clone https://github.com/Gbanya-Michael/eatclub-challenge-michael
 ```
2. Copy the environment file:
   ```bash
   cp .env.example .env
   ```
3. Add the challenge data url to the .env file
   - VITE_DATABASE_URL="eatclub-challenge-data-url"

4. Install dependencies and run the development server:
   ```bash
   npm install
   npm run dev
   ```
